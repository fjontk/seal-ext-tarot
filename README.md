# 🔮 塔罗牌AI解读 - 海豹骰插件

一个为海豹骰（SealDice）开发的塔罗牌占卜插件，支持完整78张塔罗牌，并通过 Google Gemini AI 提供智能解读。

## ✨ 功能特点

- **完整塔罗牌库**：包含全部78张塔罗牌
  - 22张大阿尔卡纳（愚者、魔术师、女祭司等）
  - 14张权杖牌组（火元素：行动、热情、创意）
  - 14张圣杯牌组（水元素：情感、关系、直觉）
  - 14张宝剑牌组（风元素：思维、沟通、冲突）
  - 14张星币牌组（土元素：物质、工作、健康）

- **两种占卜模式**
  - 🎴 **单牌占卜**：快速获取当前指引
  - 🎴 **三牌阵占卜**：过去·现在·未来的完整解读

- **AI智能解读**：使用 Google Gemini 2.0 Flash 模型，针对你的具体问题给出个性化解读

- **可爱女仆风格**：俏皮有趣的回复文案，让占卜体验更加生动

## 📦 安装方法

### 方式一：直接安装
1. 从 Release 下载编译好的 `.js` 文件
2. 在海豹骰管理面板中，进入「扩展功能」→「JS扩展」
3. 上传或粘贴插件代码

### 方式二：从源码编译
```bash
# 克隆项目
git clone <repository-url>
cd seal-ext

# 安装依赖
npm install

# 编译
npm run build
```
编译后的文件在 `dist` 目录中。

## ⚙️ 配置

使用前需要配置 Google Gemini API Key：

1. 前往 [Google AI Studio](https://aistudio.google.com/app/apikey) 获取免费的 API Key
2. 在海豹骰管理面板中，进入「扩展功能」→「JS扩展」
3. 找到 `tarot-ai` 插件，在配置项中填入 `geminiApiKey`

> 💡 **提示**：也可以直接在源码的 `HARDCODED_API_KEY` 变量中填写 API Key

## 🎮 使用方法

### 单牌占卜
```
.tarot [你的问题]
```
**示例：**
- `.tarot` - 不带问题，获取今日指引
- `.tarot 我最近的感情运势如何` - 针对具体问题占卜

### 三牌阵占卜（过去·现在·未来）
```
.tarot3 [你的问题]
```
**示例：**
- `.tarot3` - 不带问题，获取人生发展的整体解读
- `.tarot3 我和TA的关系会如何发展` - 针对具体问题占卜
